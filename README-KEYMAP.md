Molten関連のキーマップ一覧
キーマップ	コマンド	説明
<space>ro	MoltenEvaluateOperator	オペレータで評価
<space>rv	MoltenEvaluateVisual	ビジュアル選択を評価
<space>rl	MoltenEvaluateLine	現在行を評価
<space>rc	run_cell() + 保存	セルを実行して保存
<space>rm	run_and_move() + 保存	セル実行後次に移動して保存
<space>rr	MoltenReevaluateCell	セルを再評価
<space>rd	MoltenDelete	出力を削除
<space>ri	MoltenInit	カーネルを初期化
<space>rh	MoltenHideOutput	出力を非表示
<space>rs	MoltenShowOutput	出力を表示
<space>ry	copy_cell_output	出力をクリップボードにコピー ⭐新規
<space>rk	MoltenRestart	カーネルを再起動
<space>ra	run_all_cells	全セルを上から実行
<space>rb	run_cells_below	現在以下のセルを実行
